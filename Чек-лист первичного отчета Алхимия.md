# Чек-лист качества первичного отчёта «Алхимия»

Использовать перед отправкой клиенту.

## 1) Структура

- [ ] Есть все 5 блоков: Показатели, Смысл, Препараты, Рекомендации, Итог.
- [ ] В «Показателях» заполнены: Я, Здоровье, Симптом, Связь.
- [ ] В «Смысле» заполнены: формирование состояния и задача.
- [ ] В «Препаратах» заполнены: основные, дополнительно, поддержка, общий смысл.
- [ ] В «Рекомендациях» есть: основная задача, назначения по времени, зачем.

## 2) Логика

- [ ] Одна чёткая линия без альтернатив и противоречий.
- [ ] Причинно-следственная связь читается последовательно.
- [ ] «Рекомендации» прямо следуют из блока «Задача».

## 3) Язык и объём

- [ ] Текст короткий и читается за 1-2 минуты.
- [ ] Нет теории, длинных объяснений и перегруза терминами.
- [ ] Формулировки простые и конкретные.
- [ ] Тон нейтральный, без обращения на «ты/вы».

## 4) Практичность

- [ ] Назначения указаны ясно (препарат + дозировка/условие).
- [ ] Нет перегруза: 3-5 назначений максимум.
- [ ] В «Итоге» есть текущее состояние и следующий шаг.

## 5) Финальная проверка

- [ ] Нет пустых маркеров вида [значение] в финальной версии.
- [ ] Нет повторов между блоками.
- [ ] Отчёт можно понять с первого прочтения.
