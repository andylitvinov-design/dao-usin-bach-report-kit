# Алхимия отчеты

Эта папка предназначена для разработки шаблонов отчетов, которые будут отправляться клиенту в PDF.

## Основные файлы

- `Шаблон первичного отчета Алхимия.md` — базовый шаблон документа для первичного отчёта в нейтральном стиле.
- `Шаблон первичного отчета УСИН.md` — базовый шаблон документа для первичного отчёта с акцентом на анализ УСИН.
- `Чек-лист первичного отчета Алхимия.md` — проверка качества перед отправкой клиенту.

## Логика первичного отчёта

Показатели → Психосоматический смысл → Препараты → Рекомендации → Итог.

Отчёт должен быть коротким и читаться за 1-2 минуты.

## Что полезно включать в клиентский отчет

1. Короткое резюме.
   В начале отчета нужен блок на 3-5 предложений: что было сделано, какой результат получен, что это значит для клиента.

2. Понятная структура.
   Один отчет лучше строить по стабильной схеме: цель, выполненные работы, результаты, выводы, следующие шаги.

3. Язык без перегруза.
   Текст должен быть понятен клиенту без внутренних терминов команды. Если специальный термин нужен, рядом стоит дать короткое объяснение.

4. Факты и интерпретация отдельно.
   Полезно разделять: что именно произошло, какие данные это подтверждают, и какой вывод из этого делает команда.

5. Визуальная ясность.
   Для PDF-отчета важно заранее думать о таблицах, подзаголовках, акцентах, коротких списках и выносках с ключевыми результатами.

6. Практическая ценность.
   Каждый отчет должен отвечать на вопрос клиента: что из этого следует, что требует внимания, что делать дальше.

7. Следующий шаг.
   В конце нужен отдельный блок с рекомендациями: что предлагается сделать в следующем периоде, в каком порядке и зачем.
