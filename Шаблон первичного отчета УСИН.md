# Базовый шаблон документа: первичный отчёт «УСИН»

Формат: коротко, понятно, 1-2 минуты чтения.
Стиль: нейтральный, без обращения на «ты/вы».
Логика: Показатели → Психосоматический смысл → Анализ УСИН → Препараты → Рекомендации → Итог.
