# Базовый шаблон документа: вторичный отчёт «Алхимия»

Формат: коротко, понятно, 1-2 минуты чтения.
Стиль: нейтральный, без обращения на «ты/вы».
Логика: Изменения → Динамика по показателям → Препараты → Интерпретация → Рекомендации → Итог.

## ВТОРИЧНЫЙ ОТЧЁТ

Кратко: динамика состояния с прошлого отчёта и текущий фокус сопровождения.

### ЧАСТЬ 1. ИЗМЕНЕНИЯ (ПО ФАКТУ)

Период: [дата прошлого отчёта] → [текущая дата]

Что изменилось:
- [ключевое изменение 1]
- [ключевое изменение 2]
- [ключевое изменение 3]

### ЧАСТЬ 2. ДИНАМИКА ПОКАЗАТЕЛЕЙ

1) Я

Было: ~[значение]
Стало: ~[значение]

2) Здоровье

Было: ~[значение или диапазон]
Стало: ~[значение или диапазон]

3) Переход по стадии УСИН

Из: «[предыдущая стадия]» ([диапазон])
В: «[текущая стадия]» ([диапазон])

- [препарат] — [фокус/смысл]

Сейчас (актуальная схема):

- [препарат] — [фокус/смысл]
- [препарат] — [фокус/смысл]
- [препарат] — [фокус/смысл]

- [из чего вышло состояние]

Текущий этап:
- [какой этап сейчас]
- [чего пока не хватает]
- [какой навык/качество формируется]
