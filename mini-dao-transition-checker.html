Страница открыта как локальный файл (`file://`). Chrome-расширения вроде Lightshot на таких страницах часто не работают, пока для расширения не включён доступ к локальным файлам. Сейчас будет автоматическое перенаправление на локальный адрес через 2 сек. Откройте страницу через локальный адрес: http://localhost:8876/mini_dao_transition_checker.html. Если хотите оставить режим `file://`, включите в `chrome://extensions` для Lightshot опцию `Allow access to file URLs`.
